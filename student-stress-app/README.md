# Student Stress & Timetable Generator

An adaptive web ecosystem engineered to lower student academic strain by parsing free-time metrics alongside stress loads, married with quick curriculum revision mock checkpoints (CBSE, ICSE, Matriculation).

## 🚀 Architecture Overview
- **Frontend**: Vanilla HTML5, CSS3 Variables, Modulated Functional ES6 JavaScript.
- **Backend Architecture**: Decoupled Python Serverless Functions deployed on Netlify.
